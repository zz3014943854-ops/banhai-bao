# 演示操作说明

## 启动

1. 双击项目根目录中的 `启动辽传短剧.bat`。
2. 等待浏览器打开 `http://127.0.0.1:4178`。
3. 登录页账号与密码已经填好，直接点击登录。

## 推荐演示顺序

1. 首页播放焦点作品，切换选集并完成点赞、收藏和评论。
2. 返回首页，进入片库搜索“毕业”，再切换作品分类。
3. 进入创作中心，填写作品名称并提交，随后在个人中心查看投稿。
4. 从首页进入剧组招募，选择岗位并提交申请。
5. 从首页进入实训资源，预约可用场地或设备。
6. 进入消息中心打开未读消息，使用“全部已读”。
7. 进入个人中心查看收藏、申请、预约和设置。

## 数据说明

用户操作记录保存在浏览器 LocalStorage 中。刷新页面不会丢失；在“设置”中清理本地数据后可恢复初始状态。
