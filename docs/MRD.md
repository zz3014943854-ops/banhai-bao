# 《辽传短剧》市场需求文档 MRD

## 1. 市场与环境

短视频平台降低了影像创作门槛，校园创作者对短剧、竖屏叙事、AIGC 影像和纪录短片的参与度持续提升。通用视频平台适合传播成片，但不擅长承载校园内部的组队、活动与资源流程。

## 2. 用户痛点

1. 校园作品分散，难以按学院、类型和活动集中浏览。
2. 剧组招募依赖群聊，信息很快被新消息覆盖。
3. 场地设备的可用状态和申请方式不够直观。
4. 投稿资料重复填写，个人创作记录难以沉淀。
5. 互动数据与校园身份割裂，难以形成校内创作影响力。

## 3. 产品定位

“辽传短剧”定位为校园垂直影像社区与创作服务工具，区别于泛娱乐平台，重点解决“作品在哪里看、团队去哪里找、资源怎么申请、活动如何参加”四个问题。

## 4. 用户画像

### 内容观众

希望发现校内优质作品，关注校园故事与热门主创。

### 初级创作者

有课程作业或短片想法，需要寻找岗位、设备和投稿机会。

### 核心主创

持续参与剧组，需要作品展示、团队招募与个人履历沉淀。

### 跨专业协作者

来自播音、动画、表演、摄影、后期等方向，希望加入合适项目。

## 5. 场景需求

- 课间浏览近期热门短剧
- 展映周前完成投稿
- 开机前快速补齐剧组岗位
- 拍摄前预约摄影棚、录音棚和设备
- 收到审核、预约和剧组回复
- 在个人中心展示阶段性创作成果

## 6. 差异化

- 校园身份与学院信息构成可信协作基础
- 作品、岗位、活动和实训资源在同一平台联动
- 内容分类贴合传媒院校专业方向
- 交互流程针对课程创作和短周期剧组设计

## 7. 内容策略

首期片库覆盖校园青春、悬疑推理、喜剧、爱情、纪录片、动画、AIGC 和实验影像。首页通过焦点推荐、人气排行、最新投稿与活动入口组织内容。
