# 《辽传短剧》产品需求文档 PRD

## 1. 产品概述

“辽传短剧”是一款服务于辽宁传媒学院学生的校园短剧平台，覆盖作品观看、创作投稿、剧组协作、实训资源预约和消息管理。产品以移动端 Web App 形式呈现，可用于课程设计展示与交互演示。

## 2. 产品目标

1. 集中展示校内学生短剧，提升校园原创作品的可见度。
2. 缩短学生从创意到组队、拍摄、投稿的操作路径。
3. 统一呈现实训场地和设备资源，降低预约信息差。
4. 通过活动、排行和互动形成持续的校园创作氛围。

## 3. 目标用户

- 有短剧观看需求的在校学生
- 导演、编剧、演员、摄影、录音、美术、后期和制片方向学生
- 需要场地、设备或实验室支持的课程项目组
- 参与校园影像活动与作品展映的创作团队

## 4. 核心流程

### 4.1 观看流程

登录 → 首页推荐/片库搜索 → 作品详情 → 播放/选集 → 点赞/收藏/评论 → 查看主创与相似作品

### 4.2 投稿流程

创作中心 → 上传封面与视频 → 填写作品信息 → 填写主创 → 选择活动 → 提交审核 → 我的投稿查看状态

### 4.3 加入剧组流程

首页快捷入口 → 剧组招募 → 岗位筛选 → 查看详情 → 提交申请 → 我的剧组查看状态

### 4.4 资源预约流程

首页快捷入口 → 实训资源 → 类型筛选 → 选择资源 → 填写日期、时间和用途 → 提交 → 我的预约查看记录

## 5. 功能需求

### 5.1 登录

- 支持学号和手机号两种登录形式。
- 演示账号预先填入，可直接进入平台。
- 登录状态保存在 LocalStorage。

### 5.2 首页

- 展示焦点作品、活动、校园热映、人气排行和最新投稿。
- 提供投稿、剧组招募和资源预约快捷入口。
- 所有内容卡片均可进入对应详情。

### 5.3 作品片库

- 支持作品名称、主创和标签搜索。
- 支持 8 类内容筛选。
- 搜索和筛选结果实时更新。

### 5.4 播放详情

- 支持播放/暂停演示、选集切换。
- 支持点赞、收藏、评论和分享反馈。
- 展示作品简介、评分、主创信息和相似作品。

### 5.5 创作中心

- 支持封面、视频文件模拟选择。
- 收集作品名称、类型、简介、标签和主创名单。
- 支持选择投稿活动。
- 提交后生成“审核中”记录并保存至本地。

### 5.6 剧组招募

- 支持 8 类岗位筛选。
- 展示截止日期、人数、团队与岗位关键词。
- 支持查看详情和提交申请。

### 5.7 实训资源

- 展示场地、设备、机房和实验室。
- 标记可预约与使用中状态。
- 预约表单包含日期、时间、用途、人数和联系方式。

### 5.8 消息中心

- 展示系统、活动、审核、互动、预约等 8 类消息。
- 支持单条已读和全部已读。

### 5.9 个人中心

- 展示学生身份、创作者等级和数据统计。
- 汇总投稿、收藏、剧组申请、资源预约与消息记录。
- 支持设置和退出登录。

## 6. 数据策略

- 项目不连接后端接口。
- 初始内容由本地 Mock 数据提供。
- 用户操作通过 LocalStorage 保存。
- 清理浏览器数据后恢复初始演示状态。

## 7. 验收标准

- 主要页面无空白与占位内容。
- 首页核心入口均可点击。
- 搜索、筛选、点赞、收藏、评论、投稿、申请、预约和已读状态可正常反馈。
- 刷新页面后主要用户记录仍然保留。
- 适配 320px 至 440px 宽度的手机界面，并可在桌面浏览器居中演示。
