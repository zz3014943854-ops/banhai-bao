# 《辽传短剧》商业需求文档 BRD

## 1. 项目背景

校园短剧创作通常分散在课程群、社团群和个人账号中。作品传播、人员招募、资源申请与活动通知之间缺少统一入口，容易出现作品难发现、团队难组建、设备信息不透明等问题。

## 2. 业务机会

- 学校拥有稳定的短剧创作人群与专业实训资源。
- 竖屏短剧、AIGC 影像和校园纪录片带来更多创作类型。
- 校内展映、竞赛和课程汇报需要结构化的作品归档与展示平台。
- 学生希望通过作品积累个人创作履历，并找到跨专业协作伙伴。

## 3. 业务目标

1. 建立统一的校园短剧内容入口。
2. 提升校内活动投稿与资源预约效率。
3. 扩大跨学院创作协作。
4. 沉淀可持续更新的学生影像作品库。

## 4. 核心价值

### 对学生

- 更快发现作品、岗位和创作伙伴。
- 更清楚地了解活动与资源状态。
- 集中管理个人作品、申请、收藏和预约。

### 对创作团队

- 获得稳定的作品曝光和招募渠道。
- 通过评论、收藏和排行获得反馈。
- 统一维护主创名单与投稿信息。

### 对校园创作生态

- 形成作品展映、人才协作和实训资源之间的闭环。
- 覆盖影视、播音、动画、新闻等多个专业方向。

## 5. 业务边界

当前版本用于交互展示，不处理真实身份认证、视频转码、支付、版权审核和资源审批。上述能力可在后续服务端版本中扩展。

## 6. 成功指标

- 首页快捷入口到达率
- 作品播放与互动次数
- 投稿完成率
- 岗位申请完成率
- 资源预约提交率
- 消息阅读率

## 7. 阶段规划

### 第一阶段

完成移动端交互原型和本地数据闭环。

### 第二阶段

接入统一身份认证、作品上传、审核与真实资源排期。

### 第三阶段

增加创作者主页、团队协作、活动评审与数据分析。
