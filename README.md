# 辽传短剧

面向辽宁传媒学院学生的校园短剧创作、展映、协作与实训资源服务平台。

## 快速启动

优先双击 `双击打开APP.html`，无需启动服务。

如果浏览器限制本地页面，再双击 `启动辽传短剧.bat`。命令窗口需要保持开启，关闭窗口即停止 App。

演示账号和密码已显示在登录页，直接点击“登录进入平台”即可。

## 功能范围

- 校园短剧推荐、搜索、分类、排行与播放
- 点赞、收藏、评论、选集与分享反馈
- 作品上传、主创信息、活动投稿与本地记录
- 剧组岗位筛选、详情查看与加入申请
- 实训场地、设备、机房与实验室预约
- 消息已读管理与个人中心记录查看

## 技术说明

- React 18
- Vite 6
- Tailwind CSS 3
- Lucide React
- LocalStorage 本地数据持久化

项目不连接服务器，所有演示数据仅保存在当前浏览器。
